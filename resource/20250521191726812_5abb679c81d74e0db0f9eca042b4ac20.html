排列组合基础概念

排列 (Permutation)

排列关注顺序，即从n个不同元素中取出r个元素的所有可能排序方式。

公式：P(n,r) = n!/(n-r)!

示例：从5个人中选择3个人并排序坐在3个座位上，共有多少种不同的坐法？

解：P(5,3) = 5!/(5-3)! = 5!/(2!) = 5×4×3 = 60种

组合 (Combination)

组合不考虑顺序，只关注从n个不同元素中取出r个元素的所有可能子集。

公式：C(n,r) = n!/[r!(n-r)!]

示例：从5个学生中选择3名代表参加比赛，共有多少种不同的选择方法？

解：C(5,3) = 5!/[3!(5-3)!] = 5!/[3!×2!] = 10种

排列公式交互演示

计算过程:

P(5,3) = 5!/(5-3)! = 5!/2! = (5×4×3×2×1)/(2×1) = 120/2 = 60

结果: 60 种可能的排列方式

组合公式交互演示

计算过程:

C(5,3) = 5!/[3!(5-3)!] = 5!/[3!×2!] = (5×4×3×2×1)/[(3×2×1)×(2×1)] = 120/(6×2) = 120/12 = 10

结果: 10 种可能的组合方式

排列组合的应用示例

竞赛排名问题

在一场有10名参赛者的比赛中，颁发冠、亚、季军，有多少种可能的获奖组合？

解析：这是一个排列问题，因为名次顺序很重要。

P(10,3) = 10!/(10-3)! = 10!/7! = 10×9×8 = 720种不同的获奖组合

委员会组建问题

从15名教师中选出5人组成课程委员会，有多少种不同的组建方式？

解析：这是一个组合问题，因为委员会成员没有顺序之分。

C(15,5) = 15!/[5!(15-5)!] = 15!/[5!×10!] = 3003种不同的组建方式

密码组合问题

一个4位数字密码，每位可以是0-9中的任意数字，允许重复。有多少种可能的密码？

解析：这是一个有放回的排列问题。

每位有10种可能，总共有104 = 10000种不同的密码

读书顺序问题

一个学生有8本不同的书要阅读，有多少种不同的阅读顺序？

解析：这是一个全排列问题。

P(8,8) = 8! = 8×7×6×5×4×3×2×1 = 40320种不同的阅读顺序

练习题

1. 从26个英文字母中选取5个字母排成一排，有多少种不同的排列方式？

P(26,5) = 26!/(26-5)! = 26!/21! = 26×25×24×23×22 = 7,893,600种

2. 一个班级有30名学生，需要选出一个3人小组去参加科技竞赛。共有多少种不同的选择方法？

C(30,3) = 30!/[3!(30-3)!] = 30!/[3!×27!] = (30×29×28)/(3×2×1) = 4,060种

3. 有8名男生和7名女生参加一项活动，需要从中选出一个4人小组，要求小组中男女生各2名。有多少种不同的组建方式？

C(8,2) × C(7,2) = 28 × 21 = 588种

4. 一个口令由3个不同的小写字母和2个不同的数字组成，字母和数字可以任意排列。共有多少种不同的口令？

C(26,3) × C(10,2) × P(5,5) = 2,600 × 45 × 120 = 14,040,000种

知识要点总结

排列与组合的区别

  • 排列：考虑顺序的选取方式
  • 组合：不考虑顺序的选取方式
  • 关系：P(n,r) = r! × C(n,r)

常用公式

  • 阶乘：n! = n × (n-1) × (n-2) × ... × 2 × 1
  • 排列：P(n,r) = n!/(n-r)!
  • 组合：C(n,r) = n!/[r!(n-r)!] = C(n,n-r)
